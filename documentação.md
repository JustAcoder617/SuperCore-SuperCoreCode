# Pastas do projeto:
### client/
Contém outras pastas e arquivos front-end, onde tudo o que estiver lá será renderizado pelo cliente, ou seja, o navegador do usuário. O arquivo `index.html` é a porta de entrada para o front-end, e os arquivos CSS e JavaScript são usados para estilizar e adicionar interatividade à página.
# client/SuperCoreCode  
Contém o código front-end  so SuperCoreCode, parte do chatbot feito para programadores.  
## Rotas API

# /chat

Rota principal, onde nessa rota o servidor vai no arquivo .py, executa a IA, pega a resposta e envia para o solicitante nessa rota.
O projeto está em desenvolvimento, isso significa que mais rotas serão colocadas e mais poder de processamento com mais modelos.
