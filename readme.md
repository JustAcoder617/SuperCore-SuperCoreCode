# 🚀 SuperCore

### Projeto de back-end com integração de Inteligência Artificial, focado na criação de um serviço de IA chamado SuperCore, incluindo controle de termos de uso e estrutura híbrida em Node.js e Python.

# 📌 Sobre o Projeto

### O Back-End com IA é uma aplicação que:

🔎 Processa requisições envolvendo Inteligência Artificial
📡 Pode integrar com modelos locais (ex: via Ollama)
📜 Possui sistema de Termos de Uso implementado
🌐 Inclui parte cliente (HTML + CSS)
🛠 Estrutura preparada para expansão futura
🧠 SuperCore

### O SuperCore é a IA do projeto, projetada para:

* Responder perguntas
* Auxiliar na resolução de problemas
* Servir como ponte de integração entre sistemas
* Operar dentro de limites de segurança e conformidade

### A aplicação inclui regras para:

* ❌ Bloquear ações nocivas
* ❌ Impedir uso ilegal
* ❌ Restringir uso indevido da IA
* ⚠️ Limitar responsabilidade da empresa fornecedora
# 🏗️ Estrutura do Projeto
```bash
back-end-com-ia/
│
├── client/        # Parte visual (HTML + CSS)
├── main.js        # Servidor Node.js
├── main.py        # Integração/Processamento com IA
├── .gitignore
└── README.md
```

# 🛠 Tecnologias Utilizadas
* 🟢 JavaScript (Node.js) — Backend principal
* 🐍 Python — Processamento e integração com IA
* 🎨 HTML + CSS — Interface cliente
# ▶️ Como Executar

* 1️⃣ Clone o repositório
```bash
git clone https://github.com/JustAcoder617/back-end-com-ia.git
cd back-end-com-ia
```
* 2️⃣ Instale as dependências (Node.js)
```bash
npm install
```
* 3️⃣ Execute o servidor Node
```bash
node main.js
```
* 4️⃣ Execute a parte Python (se necessário)
```bash
python main.py
```

# 🔐 Segurança

### O projeto foi pensado com:

* 🕵️‍♂️ Validção de entrada
* 🚫 Regras contra uso malicioso
* 🛡️ Estrutura preparada para integração com CAPTCHA
* 📜 Termos de Uso explícitos

### O uso do sistema implica na aceitação dos Termos de Usuário Final incluídos no projeto.

### A IA pode apresentar erros ou imprecisões. O uso indevido pode resultar em bloqueio do acesso.
